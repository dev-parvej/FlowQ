== Description ==

FlowQ is a powerful WordPress plugin that revolutionizes how you collect data from your audience. Unlike traditional static surveys, FlowQ creates dynamic, interactive question flows that adapt based on user responses, providing a personalized experience that increases completion rates and engagement.

= Key Features =

* **Dynamic Question Flow** - Route users to different questions based on their answers
* **Conditional Branching** - Create complex survey logic without any coding
* **External Redirects** - Send users to specific pages based on responses
* **5 Beautiful Templates** - Classic, Modern, Card-based, Dark Mode, and Colorful
* **Participant Management** - Track responses with detailed analytics
* **Privacy-First** - GDPR-ready with built-in privacy policy support
* **Mobile Responsive** - Perfect on desktop, tablet, and mobile devices
* **Easy Integration** - Simple shortcodes work with any page builder
* **Session Management** - Track user sessions and completion rates
* **No Coding Required** - Visual builder makes it easy for everyone

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin dashboard
2. Navigate to Plugins > Add New
3. Search for "FlowQ"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin ZIP file
2. Log in to your WordPress admin dashboard
3. Navigate to Plugins > Add New > Upload Plugin
4. Choose the ZIP file and click "Install Now"
5. Activate the plugin

= After Activation =

1. You'll find a new "FlowQ" menu item in your admin sidebar
2. Click "Add New Survey" to create your first survey
3. Add questions with answer options
4. Configure conditional branching if needed
5. Copy the shortcode and paste it into any page or post

== Frequently Asked Questions ==

= Is FlowQ free to use? =

Yes! FlowQ is completely free and released under GPL v2 license.

= How many surveys can I create? =

Unlimited! Create as many surveys as you need.

= Does it work with my theme? =

Yes, FlowQ is designed to work with any properly coded WordPress theme.

= Can I use it with page builders? =

Yes! FlowQ shortcodes work with Gutenberg, Elementor, Beaver Builder, Divi, WPBakery, and all other major page builders.

= Can I create multiple choice questions? =

Currently, FlowQ supports single-choice questions. Multiple choice is planned for v1.1.

= Can I export survey responses? =

CSV export is planned for v1.1. Currently, data is viewable in WordPress admin.

= Is it GDPR compliant? =

FlowQ provides tools for GDPR compliance (privacy policies, data viewing), but you're responsible for your site's overall compliance.

= Does it slow down my site? =

No. FlowQ is optimized for performance with efficient database queries and minimal scripts.

= Can I translate FlowQ? =

Yes, FlowQ is translation-ready with text domain 'flowq'.

= Where can I get support? =

Submit issues on our GitHub repository or check the WordPress support forums.
